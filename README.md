# 🎨 Pavlovo52 Paint Bot

Telegram бот для подбора и заказа автоэмали в магазине "Автоэмали Павлово". Бот помогает клиентам подобрать краску по коду, VIN или марке автомобиля, а также отправляет уведомления администратору о новых заказах.

## ✨ Возможности

- 📝 Сбор информации о заказе (имя, телефон, марка авто, год выпуска, код краски)
- 🔍 Поиск по VIN-коду
- 🎯 Выбор приоритета (качество/цена)
- 📱 Отправка контакта через Telegram
- 📨 Автоматические уведомления администратору о новых заказах
- ✅ Валидация вводимых данных
- 🖼 Удобные inline-клавиатуры
